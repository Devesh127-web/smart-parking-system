
# Cloud-Based Multi-Tenant Smart Parking Management System

## 📌 Overview
This project is a cloud-based smart parking management system designed for shopping malls. It aims to support multiple tenants on a shared infrastructure while ensuring scalability, secure access control, and efficient resource utilization.
